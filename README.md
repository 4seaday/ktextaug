# ktextaug


Data augmentation Toolkit for Korean text.



## Installation

### Prerequisites

* Python 3.6
* Beautifulsoup4>=4.6.0
* Googletrans>=2.4.0
* Pandas>=1.0.4

in command line:

```
pip install ktextaug
```

## Getting Started

ktextaug를 사용하는 간단한 예제입니다. 

```
import ktextaug

tokenizer = bring_it_your_own # 토크나이저는 어떤 토크나이저를 사용하더라도 상관없습니다.
augmentor = kdataaug.ABC()
```
